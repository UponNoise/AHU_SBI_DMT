StatPlus Subscription

StatPlus currently offers three subscription levels: base (free), Pro ($9.99/month or $99/year) and Premium ($14.99/month or $149/year).

Base

Free
 
  • B Most of basic statistics commands*.
  • One- and two-way ANOVA.
  • Multiple linear regression with residual analysis and basic heteroscedasticity tests.
  • Rank and percentile
  • Basic time series analysis and processing commands.
  • Basic data processing and sampling commands.
  • Datasets are limited to 250 rows.

Pro

$9.99/month
$99/year
  • B All the basic statistics commands.
  • A ANOVA commands with post‑hoc tests
  • N Nonparametric statistics.
  • R Regression analysis
    (except for the Cox regression).
  • Data transformation, sampling and charting commands.
  • Export reports as Microsoft Excel documents (with charts).
  • Customer support.
  • Coming soon: access to online data analysis app.

Premium

$14.99/month
$149/year
Best Value
  • All the Pro package features.
  • Any new commands and features released during the subscription period.
  • T All the time series analysis commands.
  • S All the survival analysis commands.
  • Coming soon: Can be purchased permanently.
  • Priority customer support.
  • + Coming soon: online and desktop apps (StatPlus for Windows, StatPlus:mac).

*Complete list of commands included with the free base package:

  • Descriptive statistics (for variables in columns)
  • One- and two-sample z-test
  • Compare Means (t-test)
  • F-test for variances
  • Linear correlation and covariance
  • Histogram
  • Box-plot
  • One- and two-way ANOVA (w/o post hoc tests).
  • Multiple linear regression
    (includes residual analysis and basic heteroscedasticity tests).
  • Rank and percentile
  • Moving average
  • Exponential smoothing
  • Fast Fourier transform
  • Time series processing (mean removal, differencing).
  • Data transformation
  • Data standardization
  • Data sampling
