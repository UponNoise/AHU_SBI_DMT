1. Consider the code fragment below.

for (x = 1; x <= n; x++)
{
    a = a + x;
    for (y = 1; y <= x; y++)
         b = b + y;
}

Counting only assignment statements as operations,

(a)    exactly how many operations does this code execute in terms of n? Express this answer in closed form.

(b)   what is the order of complexity of this code in terms of n? Express this answer as a Big O function in its simplest form.
